1

Install Dependencies

cd algerian-medical-passport-blockchain
npm install
2

Start Local Blockchain

npm run node

This starts a local Hardhat blockchain network with test accounts.

3

Deploy Smart Contracts

npm run deploy:local

Deploys IdentityManagement, MedicalRecords, and PrescriptionManagement contracts.

4

Setup Demo Data

npx hardhat run scripts/setupDemo.js --network localhost

Creates sample users, medical records, and prescriptions for testing.

5

Start Applications

npm run dev

Starts both frontend (port 3000) and backend (port 3001) simultaneously.
